# MicroMVVM
## Résumé

Ce projet est un micro toolkit MVVM pour Xamarin.Forms

Ce toolkit fonctionne en .Net Standard 2.0

Il contient 

\-        Un IOC

\-        Un ViewLocator permettant de réaliser un binding automatique du viewModel sur la vue en fonction du nom de la vue

\-        UnViewModelBase qui implémente INPC

\-        Un service de navigation qui fonctionne à l’aide d’un tag qui représente le type de la vue sur laquelle naviguer

\-        Un service de dialog Popup

\-        Une classe d’extension pour XAML qui permet la localisation des ressources Textes

\-        Une classe d’extension XAML qui permet l’accès à des ressource embarqués dans la DLL .Net Standard
